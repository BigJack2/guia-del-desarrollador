JavaScript

¿Que es JavaScript?

JavaScript es un lenguaje de programación o de secuencias de comandos que te permite implementar funciones complejas en páginas web, cada vez que una página web hace algo más que sentarse allí y mostrar información estática para que la veas, muestra oportunas actualizaciones de contenido, mapas interactivos, animación de Gráficos 2D/3D, desplazamiento de máquinas reproductoras de vídeo, etc.

¿Qué hace JavaScript en tu página?

Un uso muy común de JavaScript es modificar dinámicamente HTML y CSS para actualizar una interfaz de usuario, a través de la API del modelo de objetos del documento. Ten en cuenta que el código de tus documentos web generalmente se cargan y ejecutan en el orden en que aparece en la página. Si JavaScript se carga e intenta ejecutarse antes de que se hayan cargado el HTML y el CSS al que afecta.

Un poco de historia

JavaScript desde sus comienzos no nacio como un lenguaje de escritura rigida en sus variables, por ello es un lenguaje debilmente tipado desde sus inicios.

En 2012 en Javascript no habían clases, ni módulos, el ecosistema carecía de herramientas que optimizaran el flujo de desarrollo, derivado precisamente por las carencias del lenguaje mismo.

2012 fue el año en que Typescript apareció (luego de 2 años de desarrollo), una solución de Microsoft para el desarrollo de aplicaciones con Javascript a gran escala como Bing y Office 365 despertaron en Microsoft la necesidad de una mejora a JavaScript que permitiera construir productos escalables.

Typescript es la solución a muchos de los problemas de JavaScript, está pensado para el desarrollo de aplicaciones robustas, implementando características en el lenguaje que nos permitan desarrollar herramientas más avanzadas para el desarrollo de aplicaciones.

var / let y const

JavaScript posee diferentes formas de declarar variables, esto como una solucion a su debil tipado, entre las formas posibles de hacerlo tenemos las siguientes:

Let: es un tipo de variable que no permite sobrrescribir su valor cuando esta se vuelve a declarar pero permite que se declare una variable tipo var con el mismo nombre.

Var: ya casi no se usa ya que es un tipo de variable que se puede sobreescribir a lo largo del documento, por ende no es segura aunque bien usada puede ser bastante dinamica.

Const: Es un tipo de mvariable cuyo valor no se remplaza o sobreescribe jamas,a diferencia de let este NO permite que se declare otra variable de diferente tipo pero con el mismo nombre.para const las variables van en mayuscula sostenida.

Tipos de datos

JavaScript posse tipos de datos entre los cuales los mas usados son tambien conocidos como Primitivos o Basicos y son:

String: Cadena de texto o caracteres.

Number: Enteros y decimales.

Bolean: True-False.

Operadores lógicos

Existen operadores lógicos que aplican en los bucles para hacer comparaciones y son:

igualdad en contenido , ==

igualdad en tipo de variable y contenido, ===

negación, !

«no igual que«, !=

and, &&

or, ||

>= , mayor que o igual; <, menor que

<= , menor que o igual; >, mayor que

Operaciones Básicas

Las operaciones básicas que los números admiten son:

Suma , +

Resta, –

Multiplicación, *

División, /

Operación módulo (el resto de la división) %

Expresiónes

Existen expresiones también que en realidad son abreviaciones de otras expresiones más largas y son:

x++ equivale a x = x + 1;

++x equivale a x = x + 1;

x– equivale a x = x – 1;

–x equivale a x = x – 1;

Funciones

Una función es una estructura de código que ejecuta y transforma unos inputs de entrada en un output de salida en forma de dato o acción.

function: Es una palabra reservada que precede el nombre que vamos a dar a la función para referirnos a ella.

() los paréntesis: Contienen las variables que van a almacenar los inputs que se le hayan enviado a la función (cuando llamemos una función desde un script se entenderá mejor). Es importante tener en cuenta que no es necesario que las variables que le pasemos via el script principal tengan el mismo nombre dentro de la función.

{ } las llaves: Contienen las instrucciones que va a ejecutar la función. Dentro de la función podemos programar exactamente que lo haríamos.

return: Es una variable reservada. Sólo puede devolver un valor o variable.
